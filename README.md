## ✨ Features

| Feature | Description |
|---------|-------------|
| 💸 **Transaction Management** | Add, edit, delete income & expenses with categories and notes |
| 📊 **Visual Analytics** | Interactive pie charts showing expense distribution by category |
| 🏷️ **Smart Categories** | Pre-defined categories + custom category creation |
| 🔍 **Advanced Search** | Search by title/note, filter by Today/Week/Month |
| 🌙 **Dark Mode** | Seamless theme switching with persistent preferences |
| 💾 **CSV Export** | Export all transactions to CSV for external analysis |
| 🎯 **Budget Tracking** | Set monthly budgets with spending alerts |
| 🚀 **Offline First** | 100% offline functionality using Hive database |

## 🚀 Quick Start

### Prerequisites
- Flutter SDK ≥ 3.0.0
- Dart SDK ≥ 2.17.0
- Android Studio / VS Code with Flutter extension

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/smart_expense_tracker.git

# 2. Navigate to project directory
cd smart_expense_tracker

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
